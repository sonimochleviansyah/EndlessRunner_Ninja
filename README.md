# 🎮 Endless Runner Ninja

Game sederhana bergenre **endless runner** yang dibuat menggunakan Unity.
Player akan mengendalikan karakter ninja yang terus berlari dan harus menghindari rintangan untuk mendapatkan skor setinggi mungkin.

---

## 🚀 Fitur Utama

* 🏃 Karakter berlari otomatis (endless run)
* ⬆️ Sistem lompat untuk menghindari rintangan
* 💀 Rintangan yang muncul secara terus-menerus
* 📊 Sistem skor berdasarkan jarak/waktu bertahan
* 🎮 Gameplay sederhana dan mudah dimainkan

---

## 🛠️ Teknologi yang Digunakan

* Unity Engine
* Bahasa pemrograman C#

---

## 📁 Struktur Project

* `Assets/` → Berisi semua asset game (script, sprite, scene, dll)
* `Packages/` → Konfigurasi package Unity
* `ProjectSettings/` → Pengaturan project Unity

---

## ▶️ Cara Menjalankan Project

1. Download atau clone repository ini
2. Buka Unity Hub
3. Klik **Open Project**
4. Pilih folder project ini
5. Jalankan scene utama di Unity

---

## 📌 Catatan

Folder seperti `Library`, `Logs`, dan `UserSettings` tidak disertakan karena akan dibuat otomatis oleh Unity.
